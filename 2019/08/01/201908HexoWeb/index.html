思想准备

搭建自己的博客网站从2017年(20170223,域名tingyucanhe.com)到现在也有两年多了，经历了从自己手写静态网页到使用WordPress模板再到使用Hexo。但初心一直未变，那就是找一个安静的思考的地方。之所以弃坑到Hexo，是因为WordPress越来越臃肿(各种插件一大堆)，而且编辑器越来越不友好。一个单纯码字的地方而已，用不着那么多花样，反而分散精力。

给文章添加MathJax支持

折腾了好几次，时隔一年终于弄好了（202012）。此处有两个步骤：1. 挂载mathjax脚本[5]；2. 需要修改hexo-renderer-mathjax包[6]，解决HTTPS站点默认不允许从HTTP地址获取JS脚本的问题。 npm install git+https://github.com/yusanshi/hexo-renderer-mathjax.git#old --save

不要安装 hexo-math 和 hexo-renderer-markdown-it-plus 这两个包，否则出错。
大概步骤是：1.使用Kramed代替 Marked；2.停止使用 hexo-math；3.更新 Mathjax 的 CDN 链接；4.更改默认转义规则；5.开启mathjax[4]
