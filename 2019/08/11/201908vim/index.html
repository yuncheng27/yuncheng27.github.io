Vim图册

vim常见的命令速查手册

  • vi/vim cheat sheet. PDF版本有更详细的介绍

    移动光标

    • Ctrl + y：与Ctrl + e的功能相反，光标向上走
    • Ctrl + b：屏幕向后移动一页，Ctrl + f：向前移动一页；Ctrl + u：向后移动半页，Ctrl + d：向前移动半页
    • Ctrl + o 、Ctrl + i 与 Ctrl + l 可以实现vim内标签页的左右切换
    • 连按两下 ‘ 可以调回到上一步操作的地方；连按两下z会使当前行移到屏幕中间
    • 先按数字，再按方向键，可以跳数字个间隔(上下的单位是行，左右的单位是字符)
    • $：行尾；^：行首；#l：光标移到改行第#个位置
    • 搜狗输入法的简繁切换热键是Ctrl + Shift + F

    删除文字

    • x：删除光标所在位置的后面(包括自己)#个字符
    • X：删除光标前#个字符

    复制

    • yw：将光标所在之处到字尾的字符复制到缓冲区中；#yw：复制#个字到缓冲区

    替换

    • r：替换光标所在处的字符。R：替换光标所到处的字符，直到按esc键为止

    撤销上一次操作

    • Ctrl + r：撤销的恢复

    更改

    • cw：更改光标所在处的字到字尾处
    • c#w：表示更改#个字

    跳至指定行

    • Ctrl + g：列出光标所在行号的百分比，以及总行号
    • G：移动光标至文章的第#行行首

    列出行号

    • :set nu；取消行号：:set nonu

    查找字符

    • /关键字：找到后按n会从前往后找
    • ?关键字：找到后按n会从后往前找，但按N又会反方向找

    高亮

    • 普通模式下按*会高亮与光标所在的词相同的所有词

    分屏

    • :vs 新文件或者文路径 垂直分屏；:sv 水平分屏
    • vim -o[n] file1,…filen o表示水平分屏，O表示垂直分屏，n表示分屏数
    • :only 关闭除当前屏外的其他分屏；:qa 关闭所有分屏；Ctrl + w c 关闭当前分屏

    退出vim

    • ZZ：存盘并退出(相当于:wq)；ZQ：无条件退出(相当于:q!)
    • Ctrl-z 命令将当前的 Vi/Vim 进程放到后台执行，之后 shell 环境即可为你所用；fg 命令则将位于后台的 Vi/Vim 进程放到前台执行，这样我们就再次进入 Vi/Vim 操作界面并恢复到原先的编辑状态
